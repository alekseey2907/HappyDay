ШАГ 1️⃣: ЗАПУСТИТЬ НА КОМПЬЮТЕРЕ
──────────────────────────────────────────────────────────────

   Способ А: Двойной клик на файл "start.bat"
   
   Способ Б: Открыть терминал и ввести:
             npm install
             npm run dev
             
   Открыть в браузере: http://localhost:3000
   
   ✅ Готово! Сайт работает!


ШАГ 2️⃣: РАЗМЕСТИТЬ В ИНТЕРНЕТЕ (опционально, но рекомендуется)
──────────────────────────────────────────────────────────────

   1. Зайти на: https://vercel.com
   2. Нажать "Sign Up" (регистрация)
   3. Нажать "Add New..." → "Project"
   4. Перетащить эту папку в браузер
   5. Нажать "Deploy"
   6. Подождать 2 минуты
   
   ✅ Получить ссылку типа: https://ваш-сайт.vercel.app


ШАГ 3️⃣: ОТПРАВИТЬ ВАЛЕРИИ
──────────────────────────────────────────────────────────────

   Отправить ссылку Валерии 26 октября!
   
   Пример сообщения:
   
   "Любимая моя! Поздравляю с Днём Рождения! 🎉
    Приготовил для тебя сюрприз: [ССЫЛКА]
    С любовью! ❤️"
   
   ✅ Валерия будет в восторге! 💝
